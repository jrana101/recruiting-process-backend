# Rate-them-backend

A backend app which allows to rate an interview process in IT company: HR, Tech Interview, Feedback

# Tech Stack
1. Jdk 11
2. Maven 3.6.0
3. Lombok
4. Spring Boot 2.2.6
5. Spring Data Jpa
6. MySql
7. Swagger
8. Model Mapper

# Build
`mvn clean install -P DEV`

# Run
`java -jar /target/rate-them-backend.jar`

`http://localhost:8080/`

`http://localhost:8080/swagger-ui.html`
